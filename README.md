# King's Cyber - Irwaga Centre

A professional website for King's Cyber Services at Irwaga Centre, offering a range of services including printing, photocopying, lamination, scanning, and government-related applications.

## Features

- **Responsive Design**: Mobile-friendly interface using Poppins font
- **Service Showcase**: Display of 9 key services
- **Service Request Form**: Allow customers to submit service requests
- **WhatsApp Integration**: Direct communication channel via WhatsApp
- **Contact Section**: Easy access to contact information

## Services Offered

1. Printing
2. Photocopy
3. Lamination
4. Scanning
5. KRA Services
6. NTSA Services
7. Passport Application
8. KUCCPS Applications
9. Computer Training
